Console_IO
==========

This is a demonstration program for console output and input. It was created as a central code collection for two different articles of the C++ with Qt tutorial on https://www.alternative-computer-programming.com.

You can find additional information in the related articles on the ACP website. 

The first article deals with the output functions: 
https://www.alternative-computer-programming.com/cpp-tutorial-003-console-output.html

The second article is about the input functions:
https://www.alternative-computer-programming.com/cpp-tutorial-004-console-input.html


Further parts of the tutorial can be accessed from the ACP tutorial page for the "C++ with Qt" tutorial which you can find here: https://www.alternative-computer-programming.com/cplusplus-with-qt-tutorial-index.html
